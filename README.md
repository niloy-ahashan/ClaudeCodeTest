# Tic Tac Toe

A browser-based Tic Tac Toe game built with vanilla HTML, CSS, and JavaScript.

## Features

- **Two-player mode** — take turns as X and O on the same device
- **vs Computer mode** — play against a simple AI that blocks, attacks, and picks optimal moves
- **Score tracking** — wins and draws persist across games until the page is refreshed
- **Win highlighting** — the winning combination is highlighted at the end of each game

## How to Play

Open `tictactoe.html` in any modern browser — no build step or dependencies required.

- Click a cell to place your mark
- Toggle **vs Computer** to switch between two-player and single-player modes
- Click **New Game** to reset the board (scores are preserved)
